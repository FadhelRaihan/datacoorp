Apa Itu Phishing?

Phishing adalah upaya penipuan untuk mencuri data pribadi dengan menyamar sebagai entitas tepercaya melalui email, SMS, atau situs web palsu.

Tips Melindungi Diri:

  • Jangan klik link dari SMS/email mencurigakan
  • Periksa URL situs web sebelum login
  • Gunakan autentikasi dua faktor (2FA)
  • Jika ragu, hubungi perusahaan melalui saluran resmi
